## What is S3 Storage? 

- Simple Storage Service
- Used to store and retrieve any amount, at any time, from anywhere
- Can be easily used to host a static website on the cloud 
- Provides built in redundancy by default
  - 3 Copies - each one stored across the AZs in region
- We will see we can access from AWS Console, AWS CLI, Python Boto3 
- Often the resources for an application/website are stored in S3  e.g. images, log files
- Default is files you put there that are private, you need to configure to make public 

<br>

Danger! Will delete a single file without confirmation:
- `aws s3 cp test.txt s3://tech610-montezer-first-bucket`

Danger! Will delete all files without confirmation:
- `aws s3 cp test.txt s3://tech610-montezer-first-bucket -- recursive`

Danger! Will delete all files AND the bucket itself
To delete the bucket and its contents: 
- `aws s3 rb s3://tech610-montezer-first-bucket/ --force`
